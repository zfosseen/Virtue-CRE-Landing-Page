# Virtue CRE — Canonical Domains

## ⛔ Do NOT use: `ai.virtue.cre`
Virtue does **not** own `ai.virtue.cre`, has never owned it, and will never own it. It is **not a real domain** — it was only an early working placeholder during the first build. Never put it on a banner, deck, email, profile, or any deliverable.

## ✅ Correct links to the CRE landing page
Both of these resolve to the same site:

- **`ai.virtue.company`** — clean subdomain redirect (forwards to `/cre`). Best for banners / short references.
- **`virtue.company/cre`** — the canonical path.

### Other CRE pages
- About: `virtue.company/cre/about`
- Demo: `virtue.company/cre/demo`

## Hosting
The CRE site lives as real pages under the main Virtue website (`virtue.company/cre`), deployed by Johnny on 2026-06-05. The `ai.virtue.company` subdomain redirects there so every channel lands in the same place.
